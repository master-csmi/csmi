Instructions pour les Étudiants

Cette page regroupe les rapports de stage pour l’année 2024 des Master 1 et 2 CSMI.

1. Le travail de stage

Votre travail de stage doit comprendre les éléments suivants :

  • Un rapport de stage au format PDF ou sous forme de site Antora.

  • Une présentation au format PDF.

Les documents doivent être déposés sur le site de l’année correspondante.

1.1. Comment déposer un PDF

1.1.1. Master 1

  • Cloner le dépôt GitHub correspondant :

git clone {stages-uri}
cd {repo}

  • Créer une nouvelle branche nommée m1-<votre nom> :

git checkout -b `m1-<votre nom>`

  • Ajouter votre rapport au format PDF dans le répertoire modules/m1/attachments/Nom-Prenom.pdf.

  • Ajouter votre présentation au format PDF dans le répertoire modules/m1/attachments/Nom-Prenom-slides.pdf.

  • Vérifier que les liens vers votre rapport et votre présentation sur la page master-csmi.github.io/csmi-stages-2025/csmi-stages-2025/m1/index.html sont corrects.

  • Éditer le fichier modules/m1/partials/rapports.adoc si nécessaire pour corriger la référence à votre travail.

  • Commiter vos modifications et pousser votre branche sur GitHub :

# Visualiser vos modifications
git status
# Commiter toutes les modifications
git commit -a -m"Ajout du rapport Nom Prénom"

  • Créer une Pull Request sur {pr-uri} en sélectionnant votre branche.

Le PDF de la présentation peut être déposé la veille au soir de la soutenance.

  • Une fois la Pull Request créée, vérifier que votre modification génère correctement le site via Netlify. Netlify modifie le PR et rajoute des informations sur un déploiement de test et vous permets de verifier vos modifications. Cliquer sur Deploy preview pour voir le site généré et vérifier que votre rapport et votre présentation sont bien accessibles.

Figure 1. Deploy preview Netlify: netlify a généré le site de test pour votre PR. Vous pouvez le consulter en cliquant sur le lien Deploy preview dans votre PR.

1.2. Comment déposer un rapport Antora

Pour déposer un rapport sous forme de site Antora, suivez les mêmes étapes que pour le rapport PDF, à l’exception de l’étape où vous devez déposer votre rapport PDF. À la place, ajoutez le lien vers le site Antora de votre rapport dans modules/{master}/partials/rapports.adoc.

Votre présentation pour la soutenance devra être au format PDF.
