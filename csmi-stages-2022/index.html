Master CSMI

Ceci est la page principale des rapports de stage de l’année 2022 pour les Master 1 et 2 CSMI.

1. Le travail de stage

Le travail de stage se présentera sous la forme suivante:

  • un rapport de stage au format PDF ou un site Antora

  • une présentation au format PDF

Votre travail sera déposé sur le site de l’année correspondante.

1.1. Comment déposer un PDF

1.1.1. Master 1

  • cloner le repository

git clone https://github.com/master-csmi/csmi-stages-2022
cd csmi-stages-2022

  • créer la branche m1-<votre nom>

git checkout -b `m1-<votre nom>`

  • ajouter votre rapport au répertoire modules/m1/attachments/Nom-Prenom.pdf

  • ajouter votre présentation au répertoire modules/m1/attachments/Nom-Prenom-slides.pdf

  • s’assurer que les PDF correspondent aux liens de votre rapport et présentation dans la page master-csmi.github.io/csmi-stages/csmi-stages/m1/index.html

  • éditer modules/m1/partials/rapports.adoc si besoin pour corriger la référence à votre travail

  • committer vos changements et pousser votre branche sur github

# voir vos changements
git status
# commit tous les changements
git commit -a -m"rapport nom prenom"

  • créer un Pull Request sur github.com/master-csmi/csmi-stages-2022/compare en sélectionnant votre branche

Le PDF de la présentation peut être déposé la veille au soir de la soutenance
Les stages avec rapports et soutenances confidentiels sont à envoyer par email à Christophe Prud’homme <prudhomme@unistra.fr> avec [STAGE CONFIDENTIEL] dans le sujet et les fichiers nommés comme ci-dessus.

1.2. Comment déposer un rapport Antora

Pour déposer le rapport Antora il suffit de suivre les étapes pour le rapport PDF excepté que vous n’avez pas à déposer votre rapport PDF mais à mettre le lien vers le site antora de votre rapport dans modules/{master}/partials/rapports.adoc.

votre présentation sera au format PDF.
